Artifact: Diagrama de caso de uso
Este artefacto es una representación visual de las interacciones entre los actores y el sistema en un proyecto de software
Purpose
El propósito del diagrama de casos de uso es capturar los requisitos funcionales del sistema desde la perspectiva del usuario

Description
Main DescriptionUn diagrama de casos de uso es un artefacto utilizado en proyectos de software para visualizar y describir las interacciones entre los actores y el sistema en desarrollo. Proporciona una representación gráfica de los diferentes casos de uso, que son las funcionalidades o acciones que el sistema ofrece a los actores
